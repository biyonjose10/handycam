## Project Overview

Real-time webcam visual effect for **TouchDesigner 2023 (Python 3.11)**. MediaPipe
tracks both hands; each wrist becomes a bottom corner of a dynamic triangle (apex
fixed top-center). Inside the triangle the webcam is rendered with a risograph /
cyanotype + halftone aesthetic; outside is a clean webcam passthrough with a hard edge.

Hand tracking uses the **`torinmb/mediapipe-touchdesigner`** GPU plugin. The TD
network is generated by a Python builder (a `.toe` is binary and can't be authored
as text).

## Files

| File | Purpose |
|---|---|
| `build_network.py` | Textport auto-builder; creates & wires all operators under `/handycam`. Idempotent. |
| `config.py` | Tunables: plugin paths, hand-CHOP channel names, palette, halftone, smoothing, coord flags. |
| `shaders/riso.frag` | 4-band posterize → ink palette → in-shader misregistration. |
| `shaders/halftone.frag` | Brick-grid ink-colored halftone, luminance-driven dot radius, vertical density gradient. |
| `shaders/triangle_composite.frag` | Cross-product point-in-triangle mask; paper grain + desaturation inside. |
| `docs/channel_mapping.md` | One-time discovery of the plugin's hand-CHOP channel names. |

## Key conventions / gotchas

- Shaders use TD GLSL-TOP convention: `out vec4 fragColor;`, `sTD2DInputs[]`, `vUV.st`, `uTD2DInfos[].res` (`.zw`=w,h `.xy`=1/w,1/h).
- Custom uniforms set via GLSL TOP **Vectors** page params: `vecNname`, `vecNvaluex/y/z/w` (0-indexed).
- Coordinate space: MediaPipe is top-left origin / y-down; TD UV is bottom-left / y-up. `FLIP_Y`, `MIRROR_X`, `SWAP_HANDS` in `config.py` reconcile this.
- The plugin's hand-CHOP channel names are NOT publicly documented — discover once, paste into `config.py`. The `sel_hands` Select CHOP renames them to stable `hand_left_x/y`, `hand_right_x/y`.
- Coordinate flip/mirror is baked into the GLSL uniform expressions (not a Math CHOP) for simplicity.
- `build_network.py` wraps parameter sets in a warn-on-missing helper so a single version-specific param name won't abort the whole build.
